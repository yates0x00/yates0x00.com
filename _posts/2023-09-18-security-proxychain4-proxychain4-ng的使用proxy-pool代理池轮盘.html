该工具支持http, https, socks4, socks5

还支持多种mode:

链式，随机选择， 多重代理。。。。

从 metagoofile 这个项目中学习到的。 真是踏破铁鞋无觅处啊！

1. 准备好proxy

2. 修改 /etc/proxychains4.conf, 增加内容：

[ProxyList]
# add proxy here ...
# meanwile
# defaults set to "tor
# socks5  127.0.0.1 1072
socks5  192.168.1.11 1072
socks5  192.168.1.11 1074

3. 使用：

proxychains curl xxx.me/222
