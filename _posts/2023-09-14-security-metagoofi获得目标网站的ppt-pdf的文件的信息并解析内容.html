refer to:
https://github.com/opsdisk/metagoofil

metagoofi可以从文档中获得信息, （从google查询）

安装

git clone https://github.com/opsdisk/metagoofil
cd metagoofil

pip3 install -r requirements.txt

使用

python3 metagoofil.py --help

例如

python3 metagoofil.py -d yoursite.com -t ppt

python3 metagoofil.py -d yoursite.com -t ALL  找所有

需要注意的是：貌似google没有api了？

使用 https://github.com/rofl0r/proxychains-ng 来进行IP的轮询。
